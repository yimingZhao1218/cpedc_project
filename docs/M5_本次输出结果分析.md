## 七、结论与后续建议

### 已改善点

1. 全量 qg R² 由负转正（约 0.33），60/10/30 与 val 有效点校验生效。  
2. 高产段误差由约 -37 万收窄到约 -11 万 m³/d，高产权重与 val_score 高产项起作用。  
3. 报告与 JSON/checkpoint 一致，best step 与时间戳可追溯。  
4. Early stopping 有效，best 优于 final。

### 仍存在的问题

1. **关井段**：pred_mean≈7.2e4 m³/d，零产偏高，需加强关井段损失或结构约束。  
2. **高产段**：仍系统性低估约 11 万 m³/d，可尝试进一步提高高产权重或延长 Stage C/D。  
3. **Val R²**：qg 在 val 上 R²=-0.95，val 仅 88 点，易受局部偏差影响，可关注 val 时段在时间轴上的位置及是否与 test 分布一致。

### 建议的下一步（可选）

1. **关井段**：适当加大 near-zero / 关井段损失权重，或检查 well 模型在 q≈0 时的压差下限是否过松。  
2. **高产段**：在现有「低2/中2/高3」基础上，试「低2/中2/高4」或略提高 Stage D 的 qg 权重。  
3. **步数**：当前 best 在 3800，可试 6000~8000 步并观察 best 是否后移、全量/高产 qg 是否再提升。  
4. **稳定性**：多跑 1~2 次不同 seed，看 best step 与 qg R² 的波动范围。
