### 🔴 #1 Z因子多项式 — 从附表5-2实测数据拟合

#### Z 因子最小二乘拟合过程（可复现）

**您提供的图片中的表格**即由下述过程得到。过程已写入可运行脚本与本文档，便于答辩/审计时复现。

**1. 数据准备**  
- 数据文件: `data/raw/附表5-PVT数据__2油藏流体恒质膨胀数据表.csv`（附表5-2 油藏流体恒质膨胀数据表）  
- 表结构: 第 1 行为表头，第 2 行为温度标签（16.5℃, 46.5℃, 78.0℃, 109.0℃, 140.32℃），第 3 行起为 (压力 MPa, Z@T1, Z@T2, …)。  
- 对**每个温度列**单独取 (p, Z) 实测点，例如 140.32℃ 列共 24 个点：p=12~75.732 MPa，Z=0.9966~1.3699。

**2. 拟合模型**  
- 二阶多项式：**Z(p) = a₀ + a₁·p + a₂·p²**  
- 最小二乘目标：min Σᵢ (Zᵢ − (a₀ + a₁·pᵢ + a₂·pᵢ²))²

**3. 求解方法**  
- 使用 `numpy.polyfit(p, Z, deg=2)` 得到系数 [a₂, a₁, a₀]（从高次到低次）。  
- RMSE = √(mean((Z − Z_fit)²))，用于上表“RMSE”列。

**4. 可复现代码位置**  
- **脚本**: `scripts/fit_z_factor_least_squares.py`  
  - 读附表5-2 CSV → 按温度列提取 (p, Z) → 对每列调用 `np.polyfit(p, Z, 2)` → 计算 RMSE → 打印与图片一致的多温度表格及 140.32℃ 的 a₀, a₁, a₂。  
- **运行方式**（在项目根目录）:
  ```bash
  python scripts/fit_z_factor_least_squares.py
  ```
  或指定 CSV 路径:
  ```bash
  python scripts/fit_z_factor_least_squares.py data/raw/附表5-PVT数据__2油藏流体恒质膨胀数据表.csv
  ```

**5. 代码中系数来源注释**  
- `src/pinn/torch_physics.py` 中 Z 因子块注释里已写明：多温度二阶拟合结果表（即您图片中的表格）、140.32℃ 的 a₀/a₁/a₂ 与 RMSE，并注明“拟合过程见本报告及 `scripts/fit_z_factor_least_squares.py`”。
